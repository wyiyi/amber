дәҢиҖ…жңүдҪ•еҢәеҲ«пјҹ

з®ҖеҚ•д»Ӣз»Қ

иҝҮж»ӨеҷЁпјҲFilterпјүдҫқиө–дәҺ Servlet е®№еҷЁпјҢеңЁе®һзҺ°дёҠеҹәдәҺеҮҪж•°еӣһи°ғгҖӮ
й…ҚзҪ®жҜ”иҫғз®ҖеҚ•пјҢзӣҙжҺҘе®һзҺ° Filter жҺҘеҸЈеҚіеҸҜпјҢд№ҹеҸҜд»ҘйҖҡиҝҮ @WebFilter жіЁи§Је®һзҺ°еҜ№зү№е®ҡ URL жӢҰжҲӘгҖӮ
е…·дҪ“дҪҝз”ЁжҹҘзңӢ Servletд№ӢFilter ж–Үз« гҖӮ

жӢҰжҲӘеҷЁпјҲInterceptorпјү дҫқиө–дәҺ Web жЎҶжһ¶пјҢеңЁ SpringMVC дёӯе°ұжҳҜдҫқиө–дәҺ SpringMVC жЎҶжһ¶гҖӮеңЁе®һзҺ°дёҠеҹәдәҺ Java зҡ„еҸҚе°„жңәеҲ¶гҖӮ
е®ғжҳҜй“ҫејҸи°ғз”ЁпјҢдёҖдёӘеә”з”ЁдёӯеҸҜд»ҘеҗҢж—¶еӯҳеңЁеӨҡдёӘжӢҰжҲӘеҷЁ InterceptorпјҢ дёҖдёӘиҜ·жұӮд№ҹеҸҜд»Ҙи§ҰеҸ‘еӨҡдёӘжӢҰжҲӘеҷЁ пјҢиҖҢжҜҸдёӘжӢҰжҲӘеҷЁзҡ„и°ғз”ЁдјҡдҫқжҚ®е®ғзҡ„еЈ°жҳҺйЎәеәҸдҫқж¬Ўжү§иЎҢгҖӮ

зӣёеҗҢзӮ№

жӢҰжҲӘеҷЁдёҺиҝҮж»ӨеҷЁйғҪжҳҜдҪ“зҺ°дәҶ AOP зҡ„жҖқжғіпјҢеҜ№ж–№жі•е®һзҺ°еўһејәпјҢйғҪеҸҜд»ҘжӢҰжҲӘиҜ·жұӮж–№жі•гҖӮ

жӢҰжҲӘеҷЁе’ҢиҝҮж»ӨеҷЁйғҪеҸҜд»ҘйҖҡиҝҮ Order жіЁи§Ји®ҫе®ҡжү§иЎҢйЎәеәҸгҖӮ

иҝҮж»ӨеҷЁе’ҢжӢҰжҲӘеҷЁеә•еұӮе®һзҺ°ж–№ејҸдёҚеҗҢгҖӮ

FilterпјҲиҝҮж»ӨеҷЁпјүжҳҜеҹәдәҺеҮҪж•°еӣһи°ғзҡ„гҖӮ

иҮӘе®ҡд№үзҡ„иҝҮж»ӨеҷЁиҰҒе®һзҺ° doFilter() ж–№жі•пјҢеңЁзӨәдҫӢдёҖдёӯзңӢеҲ°иҜҘж–№жі•жңү FilterChain еҸӮж•°пјҢе…¶е®һе®ғжҳҜдёҖдёӘеӣһи°ғжҺҘеҸЈгҖӮ
ApplicationFilterChain жҳҜе®ғзҡ„е®һзҺ°зұ»пјҢеңЁе®һзҺ°зұ»еҶ…йғЁжңү doFilter() ж–№жі•е°ұжҳҜеӣһи°ғж–№жі•пјҢеҸҜд»ҘиҺ·еҸ–еҲ°иҮӘе®ҡд№үзҡ„ xxxFilter зұ»пјҢ
еңЁе…¶еҶ…йғЁеӣһи°ғж–№жі• doFilter() йҮҢи°ғз”Ёеҗ„дёӘиҮӘе®ҡд№ү xxxFilter иҝҮж»ӨеҷЁпјҢ е№¶жү§иЎҢ doFilter() ж–№жі•гҖӮ
иҖҢжҜҸдёӘxxxFilter дјҡе…Ҳжү§иЎҢиҮӘиә«зҡ„ doFilter() иҝҮж»ӨйҖ»иҫ‘пјҢжңҖеҗҺеңЁжү§иЎҢз»“жқҹеүҚдјҡжү§иЎҢfilterChain.doFilter(servletRequest, servletResponse)пјҢд№ҹе°ұжҳҜеӣһи°ғApplicationFilterChainзҡ„doFilter() ж–№жі•пјҢд»ҘжӯӨеҫӘзҺҜжү§иЎҢе®һзҺ°еҮҪж•°еӣһи°ғгҖӮ

2гҖҒдҪҝз”ЁиҢғеӣҙ

FilterпјҲиҝҮж»ӨеҷЁпјүеңЁ Servlet и§„иҢғдёӯе®ҡд№үпјҢе®һзҺ° javax.servlet.Filter жҺҘеҸЈпјҢдҫқиө–дәҺ Tomcat зӯүе®№еҷЁпјҢеңЁ web зЁӢеәҸдёӯдҪҝз”ЁгҖӮ

InterceptorпјҲжӢҰжҲӘеҷЁпјүжҳҜ SpringMVC дёӯе®һзҺ°зҡ„пјҢе®һзҺ° org.springframework.web.servlet.HandlerInterceptor жҺҘеҸЈпјҢ
жӢҰжҲӘеҷЁжҳҜдёҖдёӘ Spring з»„д»¶пјҢз”ұ Spring е®№еҷЁиҝӣиЎҢз®ЎзҗҶгҖӮ

3гҖҒи§ҰеҸ‘ж—¶жңәдёҚеҗҢ

иҝҮж»ӨеҷЁFilterжҳҜеңЁиҜ·жұӮиҝӣе…Ҙе®№еҷЁеҗҺпјҢдҪҶеңЁиҝӣе…Ҙ servlet д№ӢеүҚиҝӣиЎҢйў„еӨ„зҗҶпјҢиҜ·жұӮз»“жқҹжҳҜеңЁ servlet еӨ„зҗҶе®Ңд»ҘеҗҺгҖӮ

жӢҰжҲӘеҷЁ Interceptor жҳҜеңЁиҜ·жұӮиҝӣе…Ҙ servlet еҗҺпјҢеңЁиҝӣе…ҘControllerд№ӢеүҚиҝӣиЎҢйў„еӨ„зҗҶзҡ„пјҢController дёӯжёІжҹ“дәҶеҜ№еә”зҡ„и§Ҷеӣҫд№ӢеҗҺиҜ·жұӮз»“жқҹгҖӮ

4гҖҒжӢҰжҲӘиҜ·жұӮиҢғеӣҙдёҚеҗҢ

жқЎд»¶пјҡеҗҢж—¶й…ҚзҪ®иҝҮж»ӨеҷЁе’ҢжӢҰжҲӘеҷЁпјҢзӨәдҫӢеӣӣпјҡ

зҺ°иұЎпјҡеңЁеҗҜеҠЁиҝҮзЁӢдёӯзҡ„ж—Ҙеҝ—пјҲеҰӮдёӢпјүпјҡзңӢеҲ°иҝҮж»ӨеҷЁ Filter жү§иЎҢдәҶдёӨж¬ЎпјҢжӢҰжҲӘеҷЁ Interceptor еҸӘжү§иЎҢдәҶдёҖж¬ЎгҖӮ

жү§иЎҢйЎәеәҸпјҡAFilter еӨ„зҗҶдёӯ -> Interceptor ејҖе§Ӣжү§иЎҢ -> hello controller ! -> Interceptor жү§иЎҢиҝҮзЁӢдёӯ -> Interceptor жү§иЎҢз»“жқҹ

еҺҹеӣ пјҡеӣ дёәиҝҮж»ӨеҷЁ Filter еҸҜд»ҘеҜ№жүҖжңүиҝӣе…Ҙе®№еҷЁзҡ„иҜ·жұӮиө·дҪңз”ЁпјҢ
иҖҢжӢҰжҲӘеҷЁ Interceptor еҸӘдјҡеҜ№ Controller дёӯиҜ·жұӮжҲ–и®ҝй—® static зӣ®еҪ•дёӢзҡ„иө„жәҗиҜ·жұӮиө·дҪңз”ЁгҖӮ

Interceptor еҸҜд»Ҙи®ҝй—® Action зҡ„дёҠдёӢж–ҮпјҢеҖјж ҲйҮҢзҡ„еҜ№иұЎпјҢиҖҢ Filter дёҚиғҪгҖӮ

еңЁ Action зҡ„з”ҹе‘Ҫе‘ЁжңҹйҮҢпјҢInterceptor еҸҜд»Ҙиў«еӨҡж¬Ўи°ғз”ЁпјҢиҖҢ Filter еҸӘиғҪеңЁе®№еҷЁеҲқе§ӢеҢ–ж—¶и°ғз”ЁдёҖж¬ЎгҖӮ

5гҖҒжіЁе…ҘBeanжғ…еҶөдёҚеҗҢ/ дҪҝз”Ёиө„жәҗдёҚеҗҢ

жӢҰжҲӘеҷЁжҳҜ Spring з»„д»¶пјҢеҸ— Spring е®№еҷЁз®ЎзҗҶпјҢй…ҚзҪ®еңЁ Spring ж–Үд»¶дёӯпјҢжӢҰжҲӘеҷЁеҸҜд»ҘиҺ·еҸ– IOC е®№еҷЁдёӯзҡ„еҗ„дёӘ bean пјҢ
еҲҮе…ҘзӮ№еӨҡеҰӮпјҡService еҜ№иұЎгҖҒж•°жҚ®жәҗгҖҒдәӢеҠЎз®ЎзҗҶзӯүпјҢжҺ§еҲ¶жӣҙз»ҶиҮҙгҖӮ еңЁжӢҰжҲӘеҷЁйҮҢжіЁе…ҘдёҖдёӘserviceпјҢе°ұеҸҜд»Ҙи°ғз”ЁдёҡеҠЎйҖ»иҫ‘гҖӮ

иҝҮж»ӨеҷЁеҲҷдёҚеҸҜд»ҘгҖӮ

6гҖҒжҺ§еҲ¶жү§иЎҢйЎәеәҸдёҚеҗҢ

еҪ“еҮәзҺ°еӨҡдёӘиҝҮж»ӨеҷЁжҲ–жӢҰжҲӘеҷЁеҗҢж—¶еӯҳеңЁзҡ„жғ…еҶөпјҢеёҢжңӣжҹҗдёӘиҝҮж»ӨеҷЁжҲ–жӢҰжҲӘеҷЁиғҪдјҳе…Ҳжү§иЎҢпјҢе°ұж¶үеҸҠеҲ°е®ғд»¬зҡ„жү§иЎҢйЎәеәҸгҖӮ
